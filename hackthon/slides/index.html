⚙️ Architecture

// That's it. Two lines. Your app now works offline.

const sdk = new OfflineLayer({ serverUrl: 'https://api.yourapp.com' });
await sdk.init();

// Smart fetch — network first, cache fallback
const data = await sdk.fetch('/api/messages');

// Offline action — queued & synced automatically
await sdk.queueAction('send_message', { text: 'Hello!' });

Prefetch — Silently downloads predicted content into cache before signal drops
